# Leoric

Leoric 是一个 Android 上的黑科技保活库，它可以对抗在任意 Android 版本上的 force-stop 杀进程。

> Leoric 是 Dota 中的英雄"骷髅王"，本项目名取自骷髅王的大招「重生」。
